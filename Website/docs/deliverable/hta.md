# Hierarchical Task Analysis (HTA)

P4 leads. Three HTAs — one per task. **User actions only.** Start each tree at the first *meaningful* goal. No "start interaction" or "open the app." No "system does X" — that lives in the grounding loop, not the HTA.

Self-check: every leaf begins with a verb the user performs.

## Task 1 — Learn about Piazza Duomo's heritage

```
0. Learn about Piazza Duomo's heritage
├── 1. Choose a landmark
│   ├── 1.1 Pick Duomo
│   ├── 1.2 Pick Galleria
│   └── 1.3 Pick Palazzo
├── 2. Choose a historical period
│   ├── 2.1 Set slider to medieval
│   ├── 2.2 Set slider to postwar
│   └── 2.3 Set slider to present
├── 3. Ask a question
│   ├── 3.1 Speak the question
│   └── 3.2 Type the question
└── 4. Follow up
    ├── 4.1 Tap a suggested chip
    ├── 4.2 Rephrase the question
    └── 4.3 Stop
```
