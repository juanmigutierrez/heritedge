# Marker Images — for P3b (WebXR / AR)

These images are used as **image-tracking markers** by the WebXR scene (ARScene.tsx, owned by P3).

Place the following files here before Thursday:

| File | Subject | Notes |
|---|---|---|
| `duomo-facade.jpg` | Duomo central facade | High-contrast, good for SLAM tracking. Shoot straight-on, daylight. |
| `galleria-crossing.jpg` | Galleria octagonal crossing from ground level | Include the mosaic floor detail. |
| `palazzo-reale-facade.jpg` | Palazzo Reale main facade on Piazza del Duomo | Include at least one full arch. |

## Requirements (P3b asked for these)
- Resolution: at least 1080 × 1080 px
- Format: JPEG, quality ≥ 85
- No heavy motion blur
- Avoid images with large featureless sky areas (bad for tracking)

If you are using stock photos, add attribution to `ATTRIBUTIONS.md`.
